Tribute Page

The tribute page project was my first serious project as an aspiring web developer and the requirements were provided by Free Code Camp. The Free Code Camp requirements were pretty simple and they were as follows:

  • User can view a tribute page with an image and text..
  • User can click on a link that will take me to an external website with further information on the topic.

The project requirements might have been simple, but I did a bit extra in terms of the general layout and use of Bootstrap utilities.

Personal Portfolio

The portfolio project is my second serious project as an aspiring web developer and the requirements were provided by Free Code Camp. The Free Code Camp requirements were pretty simple and they were as follows:

  • User can access all of the site’s content by scrolling.
  • User can click different buttons that lead them to various social media sites.
  • User can go to different sections of the site via a navigation bar.
  • User can see thumbnails of the various projects I’ve done.

For this project, I wanted to get more elaborate with it. I decided to learn and explore the options I had in terms of the design of the layout and color palette.
